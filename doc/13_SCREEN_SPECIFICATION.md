# 화면정의서 작성 원칙

1. 모든 CTA는 예약으로 연결한다.

2. 모든 섹션은 예약 버튼을 포함한다.

3. 모바일 사용자를 우선 고려한다.

4. 클릭 횟수를 최소화한다.

5. 신뢰 → 프로그램 → 예약 흐름을 유지한다.
