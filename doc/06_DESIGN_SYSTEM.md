# 1. 문서 목적

본 문서는 고운통증케어 홈페이지의 디자인 기준을 정의한다.

브랜드 일관성을 유지하고 사용자 경험을 향상시키기 위해

컬러, 타이포그래피, 버튼, 카드, 레이아웃 규칙을 통일한다.
