# 18. 카피라이팅 원칙

1. 전문 용어보다 이해하기 쉬운 문장 사용

2. 피부관리보다 통증관리 전문성을 함께 강조

3. 과장 표현 사용 금지

4. 고객이 얻는 결과 중심으로 작성

5. 모든 섹션은 예약 행동으로 연결
