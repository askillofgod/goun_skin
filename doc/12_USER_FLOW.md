# 1. 문서 목적

본 문서는 사용자가 홈페이지에 방문한 후

예약 또는 문의까지 진행하는 과정을 정의한다.

고운통증케어 홈페이지의 최종 목표는

"예약 전환"이다.

모든 페이지와 콘텐츠는 예약 버튼 클릭을 유도하도록 설계한다.
